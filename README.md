# Curso de Udemy de React de cero a experto usando React 19+ con TypeScript

#### Instructor: Fernando Herrera

### Contenido del curso

- React a profundidad
- Hooks y Hooks personalizados (Incluye nuevos de React 19+)
- Realizar pruebas unitarias y de integración en todo tu código
- Unit Test, Vitests
- React Testing Library
- MERN - Mongo Express React Node
- Autenticación con Google y JWTs (JsonWebTokens)
- JWT y su re-validación
- Encriptación de contraseñas
- Google Sign-in
- Autenticación personalizada
- Crear aplicaciones reales
- Redux, Context y otros manejadores de estado
- Diseños con AI
- ShadCN
- Animaciones en componentes
- TailwindCSS
- Redux DevTools, Redux ToolKit
- React DevTools
